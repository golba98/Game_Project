BIGFOOT ADVENTURE
=================

A 2D side-scrolling platformer game created by Jordan Vorster.
Guide the character through a procedurally generated landscape, 
collect coins, and avoid falling into canyons!


HOW TO PLAY
-----------
Open the "index.html" file in your web browser to start the game.

CONTROLS:
- Move Left:   Left Arrow or A
- Move Right:  Right Arrow or D
- Jump:        Spacebar or W
- Respawn:     Spacebar (press when Game Over screen appears)

OBJECTIVE:
- Run and jump to navigate the world.
- Collect the gold coins.
- Watch out for canyons! If you fall, you lose.


FEATURES
--------
- Procedural Generation: Trees and mountains are random every time.
- Physics: Gravity, jumping, and collision detection.
- Camera: Side-scrolling view that follows the player.
- Parallax Background: Clouds move slower than the foreground for depth.


INSTALLATION
------------
1. Keep "index.html", "sketch.js", and "p5.js" in the same folder.
2. Double-click "index.html" to play.
